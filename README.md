# Beach House Creatives V2

Premium AI-powered e-bike paint design studio.

## Upload to GitHub

Upload the CONTENTS of this folder. The repo root should show:

- app
- public
- package.json
- vercel.json
- README.md

## Vercel Environment Variables

Optional for demo mode:
OPENAI_API_KEY=
STRIPE_SECRET_KEY=
EMAIL_USER=
EMAIL_PASS=
TWILIO_SID=
TWILIO_AUTH=
TWILIO_PHONE=
NEXT_PUBLIC_SITE_URL=

Without keys, the app still builds and runs in demo mode.
